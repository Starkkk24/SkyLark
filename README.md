# SkyLark — Monday.com Business Intelligence Agent

An AI agent that answers founder-level business questions over monday.com **Deals** and
**Work Orders** boards — live data, deterministic metrics, conversational insights.

**The application lives in [`skylark-bi/`](./skylark-bi).** See
[`skylark-bi/README.md`](./skylark-bi/README.md) for architecture + setup, and
[`skylark-bi/DECISION_LOG.md`](./skylark-bi/DECISION_LOG.md) for assumptions & trade-offs.

## Deploy (Vercel)
Import this repo in Vercel and **set Root Directory to `skylark-bi`**, then add the environment
variables from `skylark-bi/.env.example`.
